# 🇫🇷 Traduction FR pour SCMDB – Star Citizen

Traduction française des données de **Star Citizen** pour **SCMDB** (missions, descriptions, lieux, etc.).  
Projet communautaire visant à rendre le contenu du jeu plus accessible aux joueurs francophones.

---

## 🚀 Utiliser la traduction

👉 Clique directement sur ce lien :
https://scmdb.net/?lang=https://raw.githubusercontent.com/Nagalty/scmdb-fr/main/lang-FR.json

---

## 🔧 Fonctionnement

* Basé sur le fichier du jeu (*global.ini*) et un travail de traduction communautaire existant
* Les missions, descriptions et données sont traduites en français
* L’interface de SCMDB reste en anglais
* Les variables dynamiques (ex : `[LOCATION]`, `[DESTINATION]`) sont conservées

---

## 📦 Version

* Version actuelle : **Star Citizen 4.7.1 (Live)**
* Dernière mise à jour : **22 avril 2026**

---

## ⚠️ Limites

* Certaines traductions peuvent être approximatives ou trop littérales
* Des améliorations sont en cours (ton, cohérence, terminologie)
* Quelques incompatibilités possibles lors des mises à jour du jeu

---

## 🔄 Mise à jour

À chaque nouvelle version du jeu :

1. Extraction du nouveau `global.ini`
2. Génération du fichier JSON
3. Remplacement du fichier `lang-FR.json`
4. Mise à jour de la version dans ce README

👉 Le lien reste toujours le même.
